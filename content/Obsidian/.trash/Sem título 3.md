#flashcard
Entendi. Vamos usar dois pontos (:) para separar a pergunta da resposta nos flashcards, mantendo a estrutura para o plugin de spaced repetition.  A utilização do ponto-e-vírgula (;) não é tão comum nesse contexto, mas podemos usá-lo para separar itens dentro da resposta, se necessário.

**TDAH - Grupo 1: Características Gerais**

```
O que é TDAH?: Transtorno de Déficit de Atenção e Hiperatividade, que afeta a atenção, impulsividade e hiperatividade.
<!--SR:!2023-10-27,Psiquiatria,TDAH,Conceito-->
```

```
O TDAH é um transtorno de aprendizagem?: Não, mas pode impactar o aprendizado.
<!--SR:!2023-10-27,Psiquiatria,TDAH,Aprendizagem-->
```

```
Quais áreas da vida o TDAH afeta?: Diversas áreas, incluindo acadêmica, social, familiar e profissional.
<!--SR:!2023-10-27,Psiquiatria,TDAH,Impacto-->
```

Seguindo esse padrão, você pode criar flashcards para todos os tópicos do seu resumo.  A separação com ": " torna a estrutura mais compacta, e o uso de ";" ajuda a listar múltiplos itens na resposta quando necessário.  Lembre-se de ajustar as tags para refletir os tópicos específicos de cada flashcard, facilitando a organização e revisão.

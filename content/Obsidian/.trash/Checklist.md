Claro! Aqui está um checklist com os tópicos da imagem, ideal para usar em anotações ou para controle de estudos:

### **Checklist: Urgência e Emergência / Saúde Mental**

- [x] Transtornos de Ansiedade (generalizada, pânico, fobias, TOC)
- [x] Transtornos do humor (depressão maior, bipolar)
- [x] Choque séptico e hipovolêmico
- [x] Síndromes coronarianas agudas
- [x] Transtornos somatoformes (somáticos, hipocondria), dissociativos (identidade, amnésia) e factícios
- [x] Transtornos de personalidade (paranoide, esquizoide, antissocial, borderline, histriônica)
- [x] Insuficiência cardíaca descompensada e Edema agudo pulmonar
- [x] Arritmias cardíacas
- [x] Transtornos mentais orgânicos (Delirium, Demência, Alzheimer, Parkinson)
- [x] Transtornos mentais decorrentes de substâncias psicoativas - Álcool, drogas de abuso, sedativos, hipnóticos, ansiolíticos
- [x] Acidente com animais peçonhentos (ofídicos, aracnídeos, escorpiônicos) e Choque anafilático
- [x] Cetoacidose diabética
- [x] Transtornos alimentares (bulimia, anorexia, compulsão alimentar)
- [x] AVE
